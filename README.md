# Life Loading Simulator

A philosophical, minimalist web experience that simulates the perpetual state of waiting and becoming.

## Concept

"Life is loadingâ€¦"

This is not a joke app. It is a reflection on the human condition. We are constantly in a state of preparation, optimization, and waiting for the "right moment". We expect a clean 100% completion, but life is messy, non-linear, and never truly "done".

The interface is brutalist, dark, and calm. It invites the user to stop scrolling and simply exist in the discomfort of the incomplete.

## Why it never reaches 100%

The loading bar is designed to never hit 100%. It is asymptotically close but forever out of reach.

- **Unpredictability:** Progress is not linear. It jumps, pauses, and sometimes regresses, mirroring real-life progress.
- **The limit:** It is capped at 97%.
- **The message:** There is no final destination. The process itself is the experience.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4)
- **Animation:** Framer Motion
- **Deployment:** Vercel Ready

## Features

- **Dynamic Loading Logic:** Random jumps, pauses, and regressions.
- **Time-Based Easter Eggs:**
  - **Midnight Mode (12 AM - 4 AM):** Speeds up with clarity.
  - **Morning Mode (5 AM - 9 AM):** Slows down with low motivation.
- **Rage Click Detection:** Responds to impatient users.
- **Micro-interactions:** Smooth animations, grain overlay, hover effects.
